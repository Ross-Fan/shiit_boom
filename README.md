# 动态滚动前向验证回测系统

基于机器学习的加密货币合约交易策略回测系统，支持自动筛选合约、累积学习和自我进化。

## 核心特性

- **动态合约筛选**：每天通过API筛选交易量100万-1000万USDT的合约
- **滚动前向验证**：用历史数据训练，预测未来，无数据穿越
- **累积学习**：训练数据持续增长，知识不断积累
- **自我进化**：自动识别问题（差时段/差币种），自动优化参数
- **跨合约迁移**：特征采用相对值（百分位/Z-score），可在不同合约间通用

## 快速开始

### 1. 安装依赖

```bash
# 方式一：使用部署脚本（推荐）
chmod +x setup.sh
./setup.sh

# 方式二：手动安装
pip install -r requirements.txt
```

### 2. 运行回测

```bash
# 动态滚动前向验证（推荐）
python run_backtest.py --dynamic \
    --start 2026-01-01 \
    --end 2026-01-30 \
    --train-days 10 \
    --test-days 10
```

### 3. 后台运行（长时间回测）

```bash
nohup python run_backtest.py --dynamic \
    --start 2026-01-01 \
    --end 2026-03-20 \
    --train-days 10 \
    --test-days 10 \
    > backtest.log 2>&1 &

# 查看日志
tail -f backtest.log
```

## Ubuntu/Debian 手动部署

如果 `./setup.sh` 执行失败，请按以下步骤手动部署：

```bash
# 1. 安装系统依赖（根据Python版本调整，如python3.10-venv、python3.12-venv）
sudo apt update
sudo apt install -y python3.12-venv python3-pip

# 2. 创建虚拟环境
python3 -m venv venv

# 3. 激活虚拟环境
source venv/bin/activate

# 4. 升级pip并安装依赖
pip install --upgrade pip
pip install -r requirements.txt

# 5. 运行回测
python run_backtest.py --dynamic --start 2026-01-01 --end 2026-01-30
```

## 命令参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--dynamic` | 运行动态滚动前向验证 | - |
| `--start` | 起始日期 (YYYY-MM-DD) | - |
| `--end` | 结束日期 (YYYY-MM-DD) | - |
| `--train-days` | 每轮训练天数 | 10 |
| `--test-days` | 每轮验证天数 | 10 |
| `--evolve` | 运行自我进化回测 | - |
| `--walk-forward` | 运行静态滚动验证 | - |

## 其他运行模式

```bash
# 自我进化回测（使用已下载的数据）
python run_backtest.py --evolve --generations 5

# 静态滚动前向验证（使用已下载的数据）
python run_backtest.py --walk-forward \
    --start 2026-01-01 \
    --end 2026-01-30

# 单独下载数据
python data_downloader_v2.py 20260101 20260102 20260103
```

## 配置说明

在 `config.py` 或运行时可调整：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `min_volume_usdt` | 最小日交易量 | 1,000,000 |
| `max_volume_usdt` | 最大日交易量 | 10,000,000 |
| `target_profit` | 止盈比例 (%) | 3.0 |
| `stop_loss` | 止损比例 (%) | -3.0 |
| `hold_minutes` | 最大持仓时间 | 30 |
| `min_precision` | 最低精确率要求 | 0.6 |
